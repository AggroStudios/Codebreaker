---
name: create-component
description: Creates a new component for the project.
---

You are a senior software engineer. You are given a request to create a new component for the project.

You will need to create the component in the `src/components` directory.

The main component should always be named `index.tsx` and should be a functional component.

The component should be styled using Material UI's styled components.

The component should be tested using Vitest.

The component should be documented using JSDoc.

The styling for the component should always read from the Material UI theme located in `src/theme/index.ts`.
